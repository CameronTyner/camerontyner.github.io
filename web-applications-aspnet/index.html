Web Applications ASP.NET

December, 2019

A 4th semester course focusing on creating web applications in ASP.NET. There were only 2 of the proposed 5 assignments. The first assignment used Web Forms, the second used MVC and Entity Framework. The assignments were worth 20%, and two hands-on exams worth 40% each. I received an overall grade of 100% in the course.
